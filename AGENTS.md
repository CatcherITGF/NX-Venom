# AGENTS.md

## Что это за репозиторий

NX-Venom — это дистрибутив кастомной прошивки (CFW) для Nintendo Switch, а не проект с компиляцией исходников. Репозиторий собирает готовые бинарники (`.kip`, `.nro`, `.nsp`, `.ovl`, `.bin`, `.ips`) из внешних проектов (Atmosphere, Hekate, sys-clk-oc, SaltyNX, FPSLocker и др.) вместе с INI/JSON-конфигами в zip-архивы для SD-карты Switch.

Здесь нет компиляции исходного кода, пакетного менеджера, тестового набора и линтера.

## Структура каталогов

- `Sources/NXVenom/` — основной бандл; его структура в точности повторяет корень SD-карты Switch. Пути критичны.
- `Sources/AIO/` — отдельная конфигурация для AIO Switch Updater.
- `Sources/Tools/GPU Volt/` — два автономных Python-скрипта для расчета таблиц напряжения GPU.
- `Sources/Tools/Uberhand Packages/` — дополнительные пакеты (`RAM Patch`, `TOTK UltraCam`).
- Корневые `.zip`-файлы — артефакты дистрибутива, которые генерирует CI и коммитит в репозиторий.

## Локальное рабочее пространство агентов (.agents/)

Папка `.agents/` в корне проекта зарезервирована для нужд ИИ-агентов. Она добавлена в `.gitignore` и никогда не попадает в репозиторий — содержимое только локальное.

Агент может свободно создавать внутри любые структуры данных: заметки, черновики, промежуточные результаты, кэш контекста, планы задач и т.п. Эти файлы служат для обмена информацией между сессиями и сохранения важного контекста между разными ИИ-агентами, работающими с этим проектом.

Примеры использования: `.agents/context.md`, `.agents/plans/`, `.agents/scratch/`.

## Команды сборки

«Сборка» здесь означает упаковку каталогов в zip. CI запускается только вручную через `workflow_dispatch` в `.github/workflows/build.yaml`.

Локальные эквиваленты:

```sh
# Основной бандл
cd Sources/NXVenom && zip -qqrX ../../NXVenom.zip ./

# AIO
cd Sources/AIO && zip -qqrX ../../AIO.zip ./

# SC Wizard (подмножество NXVenom)
cd Sources/NXVenom && zip -qqrX "../../SC Wizard.zip" "./switch/.packages/SC Wizard" ./switch/.overlays/ovlmenu.ovl

# Дополнительные пакеты
cd "Sources/Tools/Uberhand Packages" && zip -qqrX "RAM Patch.zip" "RAM Patch"
```

Патчи FPSLocker CI забирает из `masagrator/FPSLocker-Warehouse` — в этом репозитории они не поддерживаются.

## DSL Uberhand-пакетов

Чаще всего редактируются файлы `config.ini` внутри `Sources/NXVenom/switch/.packages/`. Это не обычный INI, а собственный DSL Uberhand:

- `[>SectionName]` — переход в подменю
- `[*SectionName]` — селектор списка
- `[@SectionName]` — только отображение информации
- `; Mariko` / `; Erista` — аппаратно-зависимые секции; у секций одно имя, а выбор определяется этим `;`-комментарием
- Команды: `kip_info`, `json_mark_cur_kip`, `hex-by-cust-offset-dec`, `set-ini-val`, `mirror_copy`, `move`, `delete`, `reboot`, `shutdown`, `catch_errors`, `back`
- JSON-файлы рядом с конфигами задают доступные значения и смещения байтов в `kip`

Не форматируй и не переставляй секции в этих файлах — порядок секций и точный синтаксис комментариев имеют значение.

## Правила редактирования

- `.editorconfig` задает: окончания строк LF, UTF-8, отступ 2 пробела, удаление хвостовых пробелов.
- Исключение: у JSON-файлов **нет финального перевода строки**. В Markdown хвостовые пробелы сохраняются.
- В путях часто есть пробелы (`SC Wizard`, `RAM Patch`, `Uberhand Packages`) — всегда заключай такие пути в кавычки в shell-командах.
- Бинарные файлы коммитятся как есть из upstream-источников. Не пытайся напрямую изменять `.kip`, `.nro`, `.nsp`, `.ovl`, `.bin` или `.ips`.

## Ограничения: что нельзя делать без явного согласия автора

Проект публичный, со строгой сложившейся структурой. Без явного запроса запрещено:

- **Редактировать `README.md`** — это витрина публичного проекта, изменения только по прямому указанию.
- **Добавлять комментарии в файлы** — не вноси пояснительные комментарии в конфиги, INI, JSON или любые другие файлы проекта.
- **Нарушать существующие паттерны структуры** — не вводи новые папки, форматы конфигов, не переименовывай секции и не меняй порядок элементов.
- **Создавать документацию** — не создавай новые `.md`-файлы, кроме `AGENTS.md`, если явно не попросили.
